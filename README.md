# WeStCOMS
Aleynik, D., 2026. WeStCOMS, Western (and Northern) Scotland Coastal Ocean Modeling System. Scottish Association for Marine Science, UK, https://doi.org/10.5281/zenodo.21249519

**We**stern (and Northern) **S**co**t**land **C**oastal **O**cean **M**odeling **S**ystem is the coupled atmosphere/ocean/waves hydrodynamic model capable of short-term 5-days forecats developed by Dr Dmitry Aleynik and run operationally at SAMS.ac.uk. It's based on WRF, FVCOM and SWAN open source code. Access the latest WeStCOMS v3 release output via: 

https://www.sams.ac.uk/facilities/thredds/ 
__&__

https://thredds.sams.ac.uk/thredds/catalog/SCOATS.html
