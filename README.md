# WeStCOMS
Western (and Northern) Scotland Coastal Ocean Modeling System is the coupled atmosphere/ocean/waves hydrodynamic model capable of short-term 5-days forecats developed by Dr Dmitry Aleynik and run operationally at SAMS.ac.uk. Its based on WRF, FVCOM and SWAN source code
Access the output via 
https://www.sams.ac.uk/facilities/thredds/ 

https://thredds.sams.ac.uk/thredds/catalog/SCOATS.html
